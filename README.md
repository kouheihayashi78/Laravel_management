# タイトル  
Laravel管理画面 

# 概要  
Laravelをもっと詳しく知りたいと考え、Laravelで管理画面を作成しました。  
投稿機能だけでなく、タグ機能を実装してシンプルなメモができるものを目指しました。  
こちらを制作して、リレーションやプロバイダー、CRUD（Create/Read/Update/Delete）などの理解を深められました。   

# URL  
http://laravel-management-kouhei.herokuapp.com

# 機能一覧 
ログイン機能  
ユーザー登録機能  
投稿機能  
タグ機能
